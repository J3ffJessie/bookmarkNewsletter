Mobile bookmarks (Chrome Sync)

Chrome extensions can't run on iOS, but Chrome's built-in bookmark sync can bridge the gap: on your phone, use Chrome's regular star/"Add bookmark" button and save the page into the folder named below. As long as you're signed into the same Google account with Chrome Sync turned on, that bookmark shows up here — click "Import now" (or use the shortcut in the popup) to pull it into your saved list.
